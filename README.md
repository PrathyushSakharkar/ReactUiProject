# Digital Banking Platform UI

A React.js UI project for a Digital Banking Platform featuring customer segmentation cards.

## Features
- Prospective customer segmentation cards (Satisfied, Unbanked, Underserved, Credit Eligible)
- Responsive horizontal card layout with images
- Clean navigation header with branding
- Reusable card components with dynamic labels

## Tech Stack
- React.js
- JavaScript
- HTML, CSS

## How to Run
1. Clone the repo
2. Run `npm install`
3. Run `npm run dev`
4. Open `http://localhost:5173`
